## CAGRについて

CAGRは、年平均複利成長率。

正式名称：

```text
Compound Annual Growth Rate
```

意味：

```text
最初の資金が最終資金になるまで、毎年平均何％で複利成長したのと同じか
```

計算式：

```text
CAGR = (最終資産 ÷ 初期資産) ^ (1 ÷ 年数) - 1
```

注意点：

```text
CAGRは途中の荒れ方を表さない。
そのため、Max DD、最悪週、最悪月、マイナス週率、マイナス月率も併せて見る。
```

## リスク別所感

### ATR OFF

```text
どのリスク設定でも最終資産・CAGRは圧倒的。
ただし、リスクを上げるとDD率も大きくなる。
```

DD目安：

```text
1.0%：Max DD 25.67%
1.5%：Max DD 36.28%
2.0%：Max DD 45.60%
```

評価：

```text
1.0%は本命候補。
1.5%は攻め候補。
2.0%は初回実運用には攻めすぎ。
```

---

### ATR P60 / Lookback 500

```text
PF_RとAvg Rは最も良い。
ただし、ATR OFFと比べてトレード数・Total R・CAGRが大きく落ちる。
```

評価：

```text
DDを抑えたい場合の保守候補。
ただし、本命ではない。
```

---

### ATR P70 / Lookback 500 現行

```text
P60よりトレード数が少なく、成長力も低い。
DD率もP60と大差ない。
```

評価：

```text
現時点では採用優先度が低い。
```

# 5. 暫定結論

## 第一候補

```text
Event Filter：
Candidate C Strategy別ハイブリッド

ATR Filter：
OFF

Risk：
1.0%
```

理由：

```text
成長力が圧倒的
マイナス月率が最も低い
マイナス週率も最も低い
月次で見ても安定性が高い
ただし、週次・月次の落ち込みは大きめ
```

---

## 第二候補

```text
Event Filter：
Candidate C Strategy別ハイブリッド

ATR Filter：
P60 / Lookback 500

Risk：
1.5%
```

理由：

```text
ATR OFFより成長力は大きく劣る
ただしDDを抑えたい場合の保守候補
```

---

## 非推奨寄り

```text
ATR P70 / Lookback 500 現行
```

理由：

```text
P60よりトレード数が少ない
成長力も低い
DD率はP60と大差ない
現時点ではP70を選ぶ理由が弱い
```

---

## 初回実運用の暫定方針

```text
まずは Event Candidate C + ATR OFF + Risk 1.0% を本命候補とする。
```

ただし、以下を理解しておく。

```text
ATR OFF 1.0%でも週単位で -10%超えがある。
月単位では最悪 -15.81% がある。
実運用では資金曲線の荒れを許容できるか確認が必要。
```
